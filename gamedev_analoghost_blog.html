w/c 10th February 2025

I have begun making space for the project page on my website. Since it is an HTML site, I had to code this in myself. In order to speed this up, I re-used elements from my previous projects for the CSS layout and the HTML baseplates.

Visual Studio Code editor for my website.

I have also begun compressing images so that the website loads faster. This is something I did not do before, and that caused pages to load a lot slower with full resolution PNG files. For this website, I have used compressed JPEG files that load fast and still are of a high enough resolution that it is legible.

w/c 3rd February 2025

After the lore was established, I was able to compose a soundtrack to match the aesthetic and atmosphere of the game. I have used Logic Pro X to do this, since it provides excellent orchestral sounds. These paired with warped tape emulation provide an unsettling feeling that aligns well with the atmosphere of the game.

Logic Pro X project file for the main music score.

The soundtrack has been made to be seamlessly loop-able so that it can serve as background music boosting the unsettling atmosphere of the game.

w/c 27th January 2025

We have sat down as a team and considered project scope and objectives. We have decided that since cutscenes are optional, we will reduce the time allocation and focus on that. In effect, we have decided to opt for a smaller cast and script. We have also changed our narrative from fully hand written to AI assisted, in order to comply with the AI usage requirements of this project.

I have created the document with the appropriate headings, and copied information from the spec sheet so that the team can begin filling it in as the project progresses.
